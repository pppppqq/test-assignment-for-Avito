# Что было реализовано?
### Задание 1
Были найдены баги на скриншоте страницы карьерного сайта Авито. Составлен подробный отчет.

Были составлены и автоматизированы тест-кейсы для проверки API сервиса, а также подготовлен баг-репорт по найденным дефектам.

# Как запустить автотесты?
Для запуска автотестов необходимо, чтобы на компьютере был установлен python и любая IDE (для удобства). После чего необходимо выполнить следующие шаги:
1. Склонировать репозиторий себе на компьютер:
   ```bash
   git clone https://github.com/pppppqq/test-assignment-for-Avito.git
   ```

3. Через командную строку/терминал перейти в корневую директорию проекта, выполнив команду  
    ```bash 
    cd /здесь укажите путь до директории с проектом/Task_2_1
    ```
4. Установить виртуальное окружение (venv):
    
    Команда для windows:
    ```bash
    python --m venv venv
    ```
    Команда для unix-систем:
    ```bash
    python3 -m venv venv
    ```

5. После чего необходимо активировать виртуальное окружение:
    
    Команда для windows:
    ```bash
    . venv/Scripts/activate
    ```
    Команда для unix-систем:
    ```bash
    source venv/bin/activate
    ```

6. После активации виртуального окружения необходимо установить зависимости из файла requirements.txt:
    ```bash
    pip install -r requirements.txt
    ```
    
7. Вот и все, теперь можете запустить тесты командой `pytest`.

# Про автотесты
Так как в процессе тестирования API были найдены баги, некоторые тесты помечены декоратором `@pytest.mark.xfail`, как заведомо провальные. Это позволяет не засчитывать их падение как ошибку, фокусироваться на новых регрессиях и документировать известные проблемы, сохраняя информативность отчётов тестирования.
